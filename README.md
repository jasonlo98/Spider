# Spider Solitaire

A browser-based implementation of Spider Solitaire developed in JavaScript.

## Web Demo
https://jasonlo98.github.io/Spider/

## Purpose
This project provides an accessible alternative to the classic Windows Spider Solitaire for platforms where native executables are not supported, such as iOS.

## Differences from the Original Project
- Increased rendering performance to 60 FPS.
- Added gameplay audio feedback for key actions.
